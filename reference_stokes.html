class pybbtd.stokes.Stokes(spatial_dims, R, L, **kwargs)[source]#

Bases: BTD

Class for tensors admitting a Stokes-constrained BTD-LL1 decomposition.

Extends BTD for polarimetric imaging: the third mode is fixed to size 4 (Stokes parameters) and each column of the spectral factor C must be a physically valid Stokes vector.

Parameters:

  • spatial_dims (list[int] or Tuple[int, int]) – Spatial dimensions (I, J) of the tensor. The full tensor has shape (I, J, 4).

  • R (int) – Number of block terms (rank of the decomposition).

  • L (int or list[int]) – Rank of the spatial maps in each term. Can be a single integer or a list of R integers.

Variables:

  • factors (list[np.ndarray] or None) – List of factor matrices [A, B, C] after fitting, or None before fitting.

  • tensor (np.ndarray or None) – Reconstructed tensor after fitting, or None before fitting.

  • fit_error (np.ndarray or None) – Array of fitting errors at each iteration, or None before fitting.

fit(data, algorithm='ADMM', **kwargs)[source]#

Fit a Stokes-BTD model to the given data using the specified algorithm.

After fitting, self.factors, self.tensor, and self.fit_error are updated in place. The ADMM solver enforces Stokes constraints on C.

Parameters:

  • data – np.ndarray Input tensor of shape (I, J, 4) to be decomposed.

  • algorithm – str Algorithm to use for fitting (default: "ADMM").

  • **kwargs – Additional keyword arguments passed to the solver (e.g. init, max_iter, rho, max_admm, rel_tol, abs_tol, admm_tol).

generate_stokes_tensor()[source]#

Generate a random Stokes tensor and store the factors.

Non-negative spatial factors A, B are drawn uniformly, and each column of C is a valid Stokes vector built from a random coherency matrix. Updates self.factors and self.tensor in place.

Returns:

A tuple (factors, tensor) where factors = [A, B, C] and tensor has shape (I, J, 4).

Return type:

Tuple[list, np.ndarray]

pybbtd.stokes.check_stokes_constraints(S)[source]#

Check if a single Stokes vector satisfies the physical constraints.

A valid Stokes vector must have \(S_0 \geq 0\) and \(S_0^2 \geq S_1^2 + S_2^2 + S_3^2\).

Parameters:

S – np.ndarray A 4-element Stokes vector [S0, S1, S2, S3].

Returns:

1 if valid, 0 if invalid.

Return type:

int

pybbtd.stokes.generate_stokes_factors(dims, R, L)[source]#

Generate random factors that follow Stokes-BTD decomposition.

Non-negative spatial factors A, B are drawn from a uniform distribution. Each column of C is a valid Stokes vector built from a random 2x2 coherency matrix.

Parameters:

  • dims (Tuple[int, int, int]) – Dimensions (I, J, 4) of the tensor.

  • R (int) – Number of block terms (rank of the decomposition).

  • L (list[int] or np.ndarray) – Rank of the spatial maps. Uses L[0] for all terms.

Returns:

A tuple (A, B, C) where:

  • A: Shape (I, L[0] * R), non-negative.

  • B: Shape (J, L[0] * R), non-negative.

  • C: Shape (4, R), each column is a valid Stokes vector.

Return type:

Tuple[np.ndarray, np.ndarray, np.ndarray]

pybbtd.stokes.proj_psd(M)[source]#

Project a matrix onto the set of positive semidefinite Hermitian matrices.

Symmetrizes M, computes its eigendecomposition, and zeroes out any negative eigenvalues.

Parameters:

M – np.ndarray Square matrix to project.

Returns:

Nearest PSD Hermitian matrix (same shape as M).

np.ndarray

pybbtd.stokes.stokes_projection(S)[source]#

Project a Stokes vector onto the set of physically valid Stokes vectors.

Converts to a coherency matrix, projects onto the PSD cone via proj_psd(), and converts back to Stokes parameters.

Parameters:

S – np.ndarray A 4-element Stokes vector [S0, S1, S2, S3].

Returns:

Projected 4-element Stokes vector satisfying the physical constraints.

Return type:

np.ndarray

pybbtd.stokes.validate_stokes_tensor(T0)[source]#

Check if all pixels in a tensor satisfy the Stokes constraints.

Each spatial pixel T0[i, j, :] is tested with check_stokes_constraints(). Emits a warning with the percentage of invalid pixels if any are found.

Parameters:

T0 – np.ndarray Tensor of shape (I, J, 4) whose third mode contains Stokes vectors.
