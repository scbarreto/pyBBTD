class pybbtd.btd.BTD(dims, R: int, L: int, block_mode='LL1')[source]#

Bases: object

Class for tensors admitting a Block Term Decomposition (BTD) into rank-(L, L, 1) terms.

The decomposition writes a third-order tensor as a sum of R block terms, each formed by the outer product of a rank-L matrix and a vector. Three block modes are supported: "LL1", "L1L", and "1LL", indicating which mode carries the rank-one structure.

Parameters:

  • dims (Tuple[int, int, int]) – Dimensions (I, J, K) of the tensor.

  • R (int) – Number of block terms (rank of the decomposition).

  • L (int or list[int]) – Rank of the block in each term. Can be a single integer (same rank for all terms) or a list of R integers.

  • block_mode (str) – Block mode of the decomposition. One of "LL1", "L1L", "1LL" (default: "LL1").

Variables:

  • factors (list[np.ndarray] or None) – List of factor matrices [A, B, C] after fitting, or None before fitting.

  • tensor (np.ndarray or None) – Reconstructed tensor after fitting, or None before fitting.

  • fit_error (np.ndarray or None) – Array of fitting errors at each iteration, or None before fitting.

check_uniqueness()[source]#

Check if sufficient conditions for essential uniqueness are satisfied.

Uses the algebraic conditions from the literature (constant-L case only). Prints whether uniqueness can be guaranteed or not. When L varies across terms, the check is skipped with a message.

After fitting, self.factors, self.tensor, and self.fit_error are updated in place.

Parameters:

  • data – np.ndarray Input tensor of shape dims to be decomposed.

  • algorithm – str Algorithm to use for fitting (default: "ALS").

  • **kwargs – Additional keyword arguments passed to the solver (e.g. init, max_iter, rel_tol, abs_tol).

get_constraint_matrix()[source]#

Return the constraint matrix for the CP-equivalent BTD model.

The constraint matrix \(\Theta\) maps the R columns of the rank-one factor to the sum(L) columns of the block factors, enabling a CP-like representation of the BTD.

Returns:

Constraint matrix of shape (R, sum(L)).

Return type:

np.ndarray

pybbtd.btd.factors_to_tensor(A, B, C, theta, block_mode='LL1')[source]#

Convert BTD factor matrices to a full tensor.

Reconstructs the tensor from factors A, B, C and the constraint matrix \(\Theta\), using the CP-equivalent representation for the given block mode.

Parameters:

  • A – np.ndarray First factor matrix.

  • B – np.ndarray Second factor matrix.

  • C – np.ndarray Third factor matrix.

  • theta – np.ndarray Constraint matrix from _constraint_matrix().

  • block_mode – str Block mode of the decomposition. One of "LL1" (default), "L1L", "1LL".

Returns:

Reconstructed tensor of shape (I, J, K).

Return type:

np.ndarray
