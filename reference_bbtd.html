class pybbtd.bbtd.BBTD(dims, R: int, L1: int, L2: int)[source]#

Bases: object

Class for tensors admitting a Block-Block Term Decomposition (BBTD) into rank-\((L_1, L_1, L_2, L_2)\) terms.

A BBTD decomposes a 4-D tensor \(\mathcal{T} \in \mathbb{R}^{I \times J \times K \times M}\) as a sum of \(R\) block terms, each of which is a rank-\((L_1, L_1, L_2, L_2)\) term built from four factor matrices A, B, C, D and two constraint matrices \(\Phi\) and \(\Psi\).

Parameters:

  • dims (list[int] or Tuple[int, int, int, int]) – Dimensions (I, J, K, M) of the 4-D tensor.

  • R (int) – Number of block terms (rank of the decomposition).

  • L1 (int) – Rank of the spatial block (shared by modes A and B).

  • L2 (int) – Rank of the spectral block (shared by modes C and D).

Variables:

  • factors (list[np.ndarray] or None) – List of factor matrices [A, B, C, D] after fitting, or None before fitting.

  • tensor (np.ndarray or None) – Reconstructed tensor after fitting, or None before fitting.

  • fit_error (np.ndarray or None) – Array of fitting errors at each iteration, or None before fitting.

check_uniqueness()[source]#

Check if sufficient conditions for essential uniqueness are satisfied.

Uses the algebraic conditions from the literature for the BBTD model. Prints whether uniqueness can be guaranteed or not.

fit(data, algorithm='ALS', **kwargs)[source]#

Fit a BBTD model to the given data using the specified algorithm.

After fitting, self.factors, self.tensor, and self.fit_error are updated in place.

Parameters:

  • data – np.ndarray Input 4-D tensor of shape matching self.dims.

  • algorithm – str Algorithm to use for fitting (default: "ALS"). Options: "ALS" (vanilla Alternating Least Squares), "ADMM" (constrained AO-ADMM with non-negativity on A, B and conjugate symmetry D = C*).

  • **kwargs – Additional keyword arguments passed to the solver (e.g. init, max_iter, rel_tol, abs_tol).

Raises:

NotImplementedError – If algorithm is not "ALS" or "ADMM".

get_constraint_matrices()[source]#

Return the constraint matrices \(\Phi\) and \(\Psi\) for the CP-equivalent BBTD model.

\(\Phi\) maps the spatial factors (A, B) and \(\Psi\) maps the spectral factors (C, D) so that the BBTD can be expressed as a CP decomposition.

Returns:

(phi, psi) where phi has shape (L1 * L2 * R, L1 * R) and psi has shape (L2 * R, L1 * L2 * R).

Return type:

Tuple[np.ndarray, np.ndarray]

pybbtd.bbtd.factors_to_tensor(A, B, C, D, phi, psi)[source]#

Reconstruct a full 4-D tensor from BBTD factor matrices and constraint matrices.

Computes A @ phi, B @ phi, C @ psi, D @ psi and assembles the CP-equivalent tensor.

Parameters:

  • A – np.ndarray Factor matrix of shape (I, L1 * R).

  • B – np.ndarray Factor matrix of shape (J, L1 * R).

  • C – np.ndarray Factor matrix of shape (K, L2 * R).

  • D – np.ndarray Factor matrix of shape (M, L2 * R).

  • phi – np.ndarray Constraint matrix for the spatial block (A, B).

  • psi – np.ndarray Constraint matrix for the spectral block (C, D).

Returns:

Reconstructed 4-D tensor of shape (I, J, K, M).

Return type:

np.ndarray
